# Indique MEI — Página de Apresentação

Este repositório contém a simulação completa do site institucional do projeto **Indique MEI**.

## 🏠 Estrutura de Arquivos
- `index.html` → Página inicial com:
  - Explicação sobre a plataforma.
  - Distribuição de taxas (Profissional, Indicador, Plataforma).
  - Seção “Gratidão pela indicação”.
  - Exemplo prático de R$ 300.
  - Botão “Entenda as Taxas”.

- `taxas.html` → Página explicativa com:
  - Modelos de cobrança e exemplos (à vista, parcelado, antecipado).
  - Tabelas com simulações reais (R$ 150 / 300 / 1.000).
  - Detalhamento de custos e lucro líquido.
  - Função de “Gratidão” destacada.

- `logo.png` → Imagem do logotipo principal (coloque aqui sua logo).

---

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (exemplo: `indique-mei`).
2. Suba os arquivos `index.html`, `taxas.html` e `logo.png` na raiz do repositório.
3. Vá em **Settings → Pages**:
   - “Build and deployment”: escolha **Deploy from a branch**.
   - “Branch”: selecione **main** e o diretório **/(root)**.
4. Clique em **Save**.
5. Após alguns minutos, o site estará disponível em:
   `https://seuusuario.github.io/indique-mei/`

---

## 💡 Personalização

- Edite textos diretamente nos arquivos `.html`.
- Substitua `logo.png` pela sua imagem oficial (mesmo nome e formato).
- As taxas e percentuais podem ser ajustadas diretamente nas tabelas.
